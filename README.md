# 🛒 AIOMarketMaker

AIOMarketMaker is an in-progress API designed to extract and structure pricing data from eBay for product valuation and analytics. It scrapes both sold and active listings, cleans and normalizes unstructured listing data, and returns structured product summaries ready for analysis.

The primary goal is to create a reliable backend service for collecting historical and live pricing data across marketplaces — starting with eBay. The system is built with modularity and extensibility in mind, using .NET 8 and Azure Functions to support scalable, on-demand data retrieval.

Support for AI-powered product disambiguation, embedding generation, and LLM-assisted entity resolution is on the roadmap. For now, development is focused on implementing and refining the core eBay scraper to ensure consistent, accurate data collection.
